# Sheryians Coding School - Landing Page Clone 

A pixel-perfect and fully responsive frontend clone of the popular [Sheryians Coding School](https://sheryians.com) landing page. This project is built using clean HTML5 and CSS3 to showcase modern frontend design principles.

🌐 *[Live Demo Link](https://lollapujitha7-cloud.github.io/Sheryians-Coding-School-Clone/)*

---

## Tech Stack Used
- *HTML5:* Semantic elements for structured and SEO-friendly web layout.
- *CSS3:* Advanced layouts using Flexbox, CSS Grid, relative units, and custom hover transitions.
- *Media Queries:* Pure CSS breakpoints ensuring 100% responsiveness across mobiles, tablets, and desktops.

---

## Key Features Implemented
- *Pixel-Perfect UI:* Exactly replicated the original dark-theme premium UI elements.
- *Fluid Responsiveness:* Navigation bar, grid items, and fonts scale dynamically according to device screen width.
- *Interactive Elements:* Smooth button transitions, card lifts on hover, and active state styles.
